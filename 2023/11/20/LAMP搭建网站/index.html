介绍

LAMP是什么？

LAMP是Linux，Apache，MySQL/MariaDB和PHP的首字母缩写，是网站管理员和开发人员用来测试和托管动态网站的一种流行的免费开源堆栈。

  • Linux 就是安装的 CentOS 7 Linux 操作系统。
  • Apache 就是一个 HTTP 服务器，负责提供 httpd 服务。
  • MySQL 就是一个关系型数据库管理系统。
  • PHP 就是一种在服务器端执行的动态脚本语言，这里是指需要安装 PHP 应用服务器。

LAMP服务器带有4个核心组件：Apache Web服务器，MySQL或MariaDB数据库以及PHP（一种流行的脚本语言，用于创建动态Web页面）。

7、创建并启动容器

docker run -id –name=ydl_tomcat -p 8081:8080 -v /root/tomcat:/usr/local/tomcat/webapps tomcat

  1. sudo docker run：这是启动一个新容器的命令，使用 sudo 权限。

  2. --name mysql57：为容器指定名称 “mysql57”。

  3. -v mysql57:/var/lib/mysql：将本地卷 mysql57 挂载到容器的 /var/lib/mysql 目录。当容器运行时，

    Docker会在主机的默认存储路径（通常是/var/lib/docker/volumes）。

  4. -e MYSQL_ROOT_PASSWORD=123456：设置 MySQL 的 root 用户密码为 “123456”。

  5. -p 3307:3306：将主机的 3307 端口映射到容器的 3306 端口。

  6. -d mysql:5.7：指定要使用的镜像（在这种情况下，是 MySQL 5.7 镜像）。

  1. sudo docker exec：这是启动一个新命令的 Docker 子命令，使用 sudo 权限。
  2. -it：这两个标志一起使用，表示以交互模式运行命令，并且将命令的输入和输出连接到容器的终端。
  3. mysql57：这是容器的名称，指定要在哪个容器中执行命令。
  4. bash：这是要在容器内部运行的命令，即启动一个 Bash 终端。
  1. sudo docker run：这是启动一个新容器的命令，使用 sudo 权限。
  2. --name phpmyadmin：为容器指定名称 “phpmyadmin”。
  3. --link mysql57:db：将容器与名为 “mysql57” 的另一个容器连接，其中 “db” 是连接的别名。
  4. -p 9998:80：将主机的 9998 端口映射到容器的 80 端口。
  5. -d phpmyadmin/phpmyadmin：指定要使用的镜像（在这种情况下，是 phpMyAdmin 镜像）。

此命令将在端口 9998 上启动一个名为 “phpmyadmin” 的 phpMyAdmin 容器，并与名为 “mysql57” 的 MySQL 容器连接。

phpMyAdmin 是一个用 PHP 编写的软件工具，可以通过 Web 方式控制和操作 MySQL 数据库。它提供了一个基于浏览器的可视化界面，让您更轻松地管理数据库。如果您需要访问和修改 MySQL 数据库，phpMyAdmin 是一个非常有用的工具。
