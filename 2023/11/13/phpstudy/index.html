下载 phpstudy

下载链接：小皮面板(phpstudy) - 让天下没有难配的服务器环境！ (xp.cn)

因为phpstudy有MySQL会与本机的MySQL有端口冲突，所以装在虚拟机上

进行配置

网站 -> 管理 ，把php版本改为5.5.9版本

打开根目录会跳转到 www 目录下，这里用来创建 index.php 等文件

点击修改可以对网站进行配置

网站 -> 修改 -> 高级配置，勾选到 目录索引 来

下载工具

下载数据库工具（客户端），方便进行管理

如果下载不了，则将 sqli.zip 在 WWW 目录下解压

首页将 Apache 和 mysql 启动，然后浏览器打开 127.0.0.1/sqli 并选择 Setup/reset Database for labs

如果出错，去检查一下php版本是否选择了5.5.9版本

如果在浏览器访问时，不知道用户名和密码，可以去MySQL的bin目录下cmd，然后去查询

启动

选择 Apache 和 mysql 启动

在虚拟机上可以直接使用 127.0.0.1 或者 localhost 去访问

如果想在本机使用，可以在虚拟机上cmd，然后ipconfig一下，将IPv4地址复制，到本机的浏览器上打开

新建文件

将根目录下的东西全删了，新建如下文件

index.php

link.php

zhuye.php

yakit

官网下载：https://yaklang.io

点击 [default]

选择 MITM 的 MIM交互式劫持 点击 免配置启动 ，进行配置
