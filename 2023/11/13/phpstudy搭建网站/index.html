介绍

实验环境：Windows。

phpstudy本地快速搭建网站，并外网访问【无公网IP】 - cpolar 极点云

下载安装

下载 小皮面板(phpstudy) ，选择Windows版本的客户端。

因为 phpstudy 中有 MySQL ，如果你的 Windows 已经安装了 MySQL ，则可能会出现端口冲突，解决方法如下：

方法一：装在虚拟机上。

方法二：在 phpstudy 中将 MySQL 的端口号改成其它的。

傻瓜式安装，唯一要注意的是安装位置。

phpstudy界面

安装后打开，出现界面如下。启动网站，启动数据库。

网站

网站域名：打开网站后，可以通过 localhost 或 127.0.0.1 进行访问。

端口：端口号一般都是 80 。

点击 管理 ：

  • 修改：可以修改网站信息。（如端口、根目录等）

    • 到 高级配置 下面勾选到 目录索引 来。（这里可以输入网站首页，一般建议为： index.php index.html ）

  • php版本：可以选择其它版本的php。（这里我选择把php版本改为5.5.9版本）

  • 打开网站：会打开浏览器，跳转到 localhost 。

  • 打开根目录：会跳转到 WWW 目录，这是网站的根目录。你可以去访问看看，也可以将根目录下的东西全删了，之后用来创建后缀为 .php、**.html** 的文件。

数据库

里面可以看见数据库用户名和密码。

软件管理

可以下载一些软件。点击 管理 会跳转。（这里我下了一个 phpMyAdmin 用于管理数据库）

phpMyAdmin 是一个以 PHP 为基础，以Web-Base方式架构在网站主机上的 MySQL 的数据库管理工具，让管理者可用Web接口管理MySQL数据库。

根目录

打开根目录，进入名为 WWW 的文件夹，可以把全部删掉，然后新建一个 index.php 文件，然后在软件管理下载的 phpMyAdmin 也会出现在这里。

在 index.php 文件中编写一些 php 和 html 语法。打开网站后，效果如下所示：

启动

在 首页 选择 Apache 和 mysql 启动。

在本地可以直接使用 127.0.0.1 或者 localhost 去访问。

如果想在其它地方使用，可以在虚拟机上 cmd ，然后 ipconfig 一下，将IPv4地址复制到其它地方的浏览器上打开访问。

PHP简介

PHP 是一种创建动态交互性站点的强有力的服务器端脚本语言。

学习PHP应当具备的基础知识：HTML、CSS

  • PHP（全称：PHP：Hypertext Preprocessor，即”PHP：超文本预处理器”）是一种通用开源脚本语言。
  • PHP 脚本在服务器上执行。
  • PHP 可免费下载使用。
  • PHP 文件可包含文本、HTML、JavaScript代码和 PHP 代码
  • PHP 代码在服务器上执行，结果以纯 HTML 形式返回给浏览器
  • PHP 文件的默认文件扩展名是 .php。

变量

PHP 变量规则：

  • 变量以 $ 符号开始，后面跟着变量的名称
  • 变量名必须以字母或者下划线字符开始
  • 变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ）
  • 变量名不能包含空格
  • 变量名是区分大小写的（$y 和 $Y 是两个不同的变量）
  • PHP 没有声明变量的命令。
  • 变量在您第一次赋值给它的时候被创建

PHP 是一门弱类型语言

PHP 有四种不同的变量作用域：local、global、static、parameter

在所有函数外部定义的变量，拥有全局作用域。除了函数外，全局变量可以被脚本中的任何部分访问，要在一个函数中访问一个全局变量，需要使用 global 关键字。

echo 和 print

echo 和 print 区别:

  • echo - 可以输出一个或多个字符串。
  • print - 只允许输出一个字符串，返回值总为 1。

提示：echo 输出的速度比 print 快， echo 没有返回值，print有返回值1。

echo 是一个语言结构，使用的时候可以不用加括号，也可以加上括号： echo 或 echo()。print 同样是一个语言结构，可以使用括号，也可以不使用括号： print 或 print()。

数据类型

PHP 变量存储不同的类型的数据，不同的数据类型可以做不一样的事情。

PHP 支持以下几种数据类型:

  • String（字符串）
  • Integer（整型）
  • Float（浮点型）
  • Boolean（布尔型）
  • Array（数组）
  • Object（对象）
  • NULL（空值）
  • Resource（资源类型）

类型比较

详细类型比较：(PHP 类型比较 | 菜鸟教程 (runoob.com))

虽然 PHP 是弱类型语言，但也需要明白变量类型及它们的意义，因为我们经常需要对 PHP 变量进行比较，包含松散和严格比较。

  • 松散比较：使用两个等号 == 比较，只比较值，不比较类型。
  • 严格比较：用三个等号 === 比较，除了比较值，也比较类型。
