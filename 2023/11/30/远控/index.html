远控

Posted on 2023-11-30 Edited on 2024-01-07 In 远控
如何远程操控主机，包括大灰狼(DHLC)远控、Quasar远控、AhMyth安卓远控、自解压病毒、修改.exe文件的图标。

Windwos防火墙关闭

有防火墙在不是很方便，比如其它电脑ping本机时会被阻拦，或者进行实验时会被防火墙干扰，所以需要把防火墙关闭。

可以运行 StopDefender.exe 进行关闭防火墙，这样更方便。

但是可能会不管用。

大灰狼(DHLC)远控

解压文件

解压 DHLv9.5.zip 文件。（注意：要关闭实时保护才能运行，不然会被删掉。）

点击打开 大灰狼远程管理软件 V9.5.exe 。打开界面如下：

生成远控文件

点击系统设置，查看需要进行监听的端口号是什么，一般为 8000 。但如果你的电脑上 8000 端口已经被使用，则需要修改成其它的端口号。

点击 服务生成 ，上线信息中的 域名IP 填写本机的IP，填写后可以点击旁边的 测试 按钮进行测试。

最后点击 生成服务 按钮，会生成一个可执行文件 .exe。

将生成的可执行文件放到目标机上运行。

这里我的目标机是VMware里面的一台桥接模式的虚拟机。

注意：要关闭实时保护才能运行，不然会被删掉。

Quasar远控

解压文件

解压 Quasar最新内部成员汉化版远控工具.rar 文件。

注意：如果出现 被系统自动删掉；被系统隔离了；提示无法成功完成操作，因为文件包含病毒或浅在的垃圾软件；等情况，建议关闭实时保护。）

进入后会发现两个文件，一个 Quasar.exe 是英文界面，一个 Quasar_zh-CHS.exe 是中文界面，这里我使用 Quasar.exe 进行演示。

点击打开 Quasar.exe 。打开界面如下：

生成远控文件

在工具栏中点击 Settings ，出现界面后，再点击 start listening ，再点击下面的 save 后关闭。

在工具栏中点击 Builder ，出现界面后。

在左侧选择 Basic，右侧点击 Random Mutex ，在下面勾选。

在左侧选择 Connection ，右侧输入 IP ，点击 Add Host 。

在左侧选择 Installation ，勾选 Install Client，勾选 Run 。

最后右下方 点击 Build Client 。就会提示生成成功。

远程操控

当生成的远控exe文件被执行后，你的 Quasar 里面就会显示新上线的主机信息，并且右击可以查看功能。

AhMyth安卓远控

下载地址：https://github.com/AhMyth/AhMyth-Android-RAT/releases

解压 AhMyth_Win64.zip 文件后，点击 AhMyth_Win64.exe 就会自动安装，不经过我们的同意。

这个是对安卓进行远控，也就是对手机端进行远控，但是如果不想用自己的手机，可以用安卓模拟器 MuMu模拟器（选择 MuMu模拟器6 的， MuMu模拟器12 的好像是打游戏的，不太适合）。

步骤一：APK输入本机IP和端口号，点击 Build 生成，找到生成文件地址

步骤二：输入端口号，点击 Listen 进行监听。

步骤三：在 MuMu模拟器 中安装，打开运行，Open The Lab

步骤四：可以进行远控了

Winrar

WinRAR下载地址：https://www.winrar.com.cn/、https://www.win-rar.com/start.html?&L=7

可以在Winrar中进行压缩

步骤一：设置压缩文件格式为 .exe，并于其它文件进行合并压缩

步骤二：点击高级设置，选择自解压路径，选择解压后自运行，选择隐藏运行

改图标

解压 Restorator.rar 文件，然后运行 Restorator.exe 文件。

将两个exe文件拖进去，然后拖动图标进行更换

但是改完图标后还是可能会出现问题，比如在桌面上看是正常的，但是移动到文件夹里就会变成原来的样子
