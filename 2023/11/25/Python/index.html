    Python介绍

    编程语言分类

    1. 编译型：C，go
      • 一次性把源代码进行翻译
      • 特点：执行效率很高，平台依赖性一般比较高
    2. 解释型：javascript，python，ruby
      • 逐行的对源代码进行翻译和运行
      • 特点：执行效率相对较低，平台的依赖性不高

    Python语言文件类型

    (1)源代码文件(source file)：.py 的源代码文件，由Python解释器负责解释执行，可在控制台下运行；.pyw的源代码文件，用于图形用户界面(GUI)文件的扩展名。

    (2)字节码文件(byte-code file)：Python源文件经过编译之后生成扩展名为 .pyc 的字节码文件。字节码文件不能用文本编辑器打开。字节码文件与平台无关，可以运行在Windows、UNIX和Linux等操作系统上。通过运行脚本，可以将. py源代码文件编译成.pyc字节码文件。

    (3)优化字节码文件(optimized file)：经过优化的代码生成扩展名为 . pyo 的优化字节码文件。该类文件不能用文本编辑器打开或者编辑。Python 3.5版本开始采用. pyc文件存储优化和非优化代码，不再支持. pyo文件

    Python下载安装

    Python语言解释器是一个轻量级的小软件，可以在Python语言主网站上下载，下载网址为：python下载 、Windows下载python （Stable Releases是稳定版本，Pre-releases是预发布版本）。

    如果要下载，请下载 Python 的 3.0 版本。这版本常被称为 Python 3000，或简称 Py3k。相对于 Python 的早期版本，这是一个较大的升级。为了不带入过多的累赘，Python 3.0 在设计的时候没有考虑向下兼容。重要的是官方宣布，2020 年 1 月 1 日， 停止 Python 2 的更新。

    操作 含义
    str.title() 返回新的字符串，字符串中每个单词的首字母大写
    str.capitalize 返回新的字符串，字符串首字符大写
    str.upper() 返回新的字符串，字符串中字母全部大写
    str.lower() 返回新的字符串，字符串中字母全部小写
    str.swapcase() 返回新的字符串，字符串中大小写互换

    操作 含义
    str.isupper() 当str所有字符都是大写，返回True，否则False
    str.islower() 当str所有字符都是小写，返回True，否则False
    str.startswith(sub) 判断字符串str是否以指定字符串sub开始
    str.endswith(sub) 判断字符串str是否以指定字符串sub结束
    str.isalnum() 测试字符串str是否为数字或字母
    str.isalpha() 测试字符串str是否为字母
    str.isdigit() 测试字符串str是否为数字字符
    str.isdecimal() 测试字符串str是否为数字(不支持汉字数字、罗马数字)字符
    str.isnumeric() 当str所有字符都是数字(支持汉字数字、罗马数字)字符时，返回True，否则False
    str.isspace() 当str所有字符都是空格，返回True，否则False
    str.isprintable() 当str所有字符都是可打印的，返回True，否则False

    操作 含义
    str.find(sub) 用来查找sub串在字符串str指定范围(默认是整个字符串)中首次和最后一次出现的位置，如果不存在则返回-1
    str.sount(sub) 返回sub子串出现的次数

    操作 含义
    str.split(sep=None) 返回新的字符串，返回一个根据sep分割后的列表(数组)，默认空格
    str.join(iter) 返回新的字符串，将iter变量的每一个元素后增加一个str字符串
    str.strip([chars]) 返回新的字符串，从str中去掉其左右两侧的chars字符。如果不指定chars参数，则默认去除字符串开头和结尾的空格字符。
    str.replace(old,new) 返回新的字符串，将str中的old字符用new字符替换

    \W 匹配任意非字母、非数字以及非下划线，等价于[^a-zA-Z0-9]

    ^ 匹配行首，匹配以^后面的字符开头的字符串
    $ 匹配行尾，匹配以$之前的字符结束的字符串
    \b 匹配单词头或单词尾
    \B 匹配非单词边界，与\b相反
    \A 匹配字符串开始，忽略多行模式
    \Z 匹配字符串结束，忽略多行模式

    几何布局管理器

    tkinter几何布局管理器( geometry manager),提供了特定的几何状态管理方法，管理整个控件区域组织，即组织和管理父组件中子组件的摆放方式。tkinter 提供了3种不同风格的几何布局管理类: pack、grid 和place。

    pack几何布局管理器

    pack几何布局管理器采用“块”的方式组织组件。pack布局根据子组件创建生成的顺序，将其放在快速生成界面设计中，因此被广泛采用。

    调用子组件的方法pack(),则该子组件在其父组件中采用pack布局：pack(option = value, …)

    pack方法提供参数选项如下:

    grid几何布局管理器

    grid几何布局管理器采用表格结构组织组件。子组件的位置由行/列确定的单元格决定，子组件可以跨越多行/列。每一列中，列宽由这一- 列中最宽的单元格确定。grid 布局适合于表格形式，可以实现复杂的界面，因此被广泛采用。

    调用子组件的grid( )方法，则该子组件在其父组件中采用grid几何布局：grid( option = value, … )

    grid方法提供参数选项如下:

    grid有两个重要的参数，就是row和column,用来指定将子组件放置到什么位置，如果不指定row,会将子组件放置到第一个可用的行上，如果不指定column则使用第0列(首列)。

    place几何布局管理器

    turtle 模块绘图

    turtle库也被称为”海龟绘图”，是Python语言中一个很流行的绘制图像的函数库，在一个横轴为x、纵轴为y的坐标系原点(0, 0)位置开始，它根据组函数指令的控制， 在这个平面坐标系中移动进行图形绘制。

    这里会使用两种坐标系:屏幕坐标系和笛卡儿坐标系。

    • 屏幕坐标系是指电脑或者手机屏幕的坐标系，单位为像素。屏幕坐标系的X轴从左往右递增，Y轴从上往下递增，X轴和Y轴没有负方向，即屏幕的最左上方可见像素的坐标是(0, 0)。X、Y的可视值域取决于显示器分辨率。以1024x768的分辨率为例，X的值在01024, Y的值在0768，只要超出这个范围，我们就看不见了。

    • 绘制的时候所使用的坐标是数学中的笛卡儿坐标，初始原点在画布正中心，向右为X轴正方向，相反方向为负，向上为Y轴正方向，相反方向为负。

    turtle库包含了100多个功能函数，主要包括窗体函数、画笔状态丽数和画笔运动函数。

    1. 窗体函数。用于展开一个画布， 用来设置绘图区域的大小和初始位置。
      • 语法格式：turtle.screensize(canvwidth,canvheight,bg)
        • canvwidth：窗口的宽度，单位像素。默认400
        • canvheight：窗口的高度，单位像素。默认300
        • bg：窗口的背景颜色
        • 例如：turtle. screensize( 800, 600,” green” )。可以设置大小为800x600,颜色为绿色
      • 语法格式：turtle.setup( width, height, startx, starty )
        • width, height：窗口的宽度和高度。输人值为整数时，单位为像素;输人值为小数时，表示窗口宽度与电脑屏幕的比例。
        • startx, starty：这一坐标表示矩形窗口左上角顶点的位置，如果为空，则窗口位于屏幕中心。
        • 例如：turtle.setup( width=0.5, height=0.5, startx=200, starty=200 )。turtle.setup( 600, 400, 0, 0 )。turtle.setup( 600, 400, )
