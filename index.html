Proxifier

介绍

1）支持 Windows 和 MacOS 系统，Windows XP/Vista/Win7/Win10，MacOS，支持 http/https、socks4/5 等代理协议，支持TCP、UDP协议

2）Proxifier 代理客户端设置后，所有网络应用程序都可以实现代理上网，不需要逐一设置代理

3）Proxifier 代理客户端开启后，所有网络应用程序必须取消代理设置，否则客户端代理上网会失败

4）通过Proxifier代理上网，客户端必须先启动Proxifier，才能访问互联网

5）通过代理服务器解析 DNS 名称，通过代理服务器链可使用不同的协议

简单来说，就是控制哪些软件应用使用代理，是否使用系统代理。感觉和自己去Clash打开代理没什么区别。

使用

0、打开proxifier后，界面如下：

1、打开 代理服务器/Proxy Servers 的窗口，点击 添加 按钮。

2、输入代理服务器 地址/Address 和 端口/Port，选择协议，如果不知道怎么选，就选择 SOCKS Version 5。

127.0.0.1：这个填的是代理服务器的地址，因为我本机就有代理软件Clash，所有填本机的IP。但是每次断网重连的时候，都有可能被分配到一个新的IP地址，所以填127.0.0.1。127.0.0.1就是本机的意思，本机回环地址。

7890：是我代理软件Clash软件的端口号。具体要看看你自己的端口号，也有可能是别的，不一定是7890。

socks 5：看情况选择协议。也可以选择socks 4、https。

若代理IP没有密码，则不用授权用户名、密码。若代理ip有密码，则设置授权的用户名、密码，然后点击“检查”。若提示测试已通过则说明配置正确可以使用。

3、在代理服务器设置界面点击 检查 ，会对你填写代理服务器进行检查。（这步可有可无）
